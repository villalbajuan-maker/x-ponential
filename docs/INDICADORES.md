# Indicadores iniciales del equipo

Estos indicadores todavia no son metricas definitivas. Son senales para observar si la dinamica del equipo se esta volviendo sana, productiva y sostenible.

## Salud relacional

- Las conversaciones dificiles se pueden tener sin deteriorar la confianza.
- Las diferencias de criterio producen mejores decisiones.
- Las personas sienten que su aporte es escuchado y exigido.

## Calidad de decision

- Las decisiones importantes tienen contexto, responsable y siguiente paso.
- El equipo distingue entre opinion, hipotesis, evidencia y acuerdo.
- Las decisiones se revisan cuando aparece nueva informacion relevante.

## Compromiso operativo

- Los compromisos se cumplen o se renegocian temprano.
- Hay responsables claros para cada avance importante.
- El seguimiento no depende de memoria informal.

## Velocidad de aprendizaje

- Las ideas se convierten en experimentos pequenos.
- El equipo aprende aun cuando una iniciativa se descarta.
- Las propuestas mejoran con evidencia, no solo con entusiasmo.

## Construccion de activos

- El trabajo deja capacidades reutilizables.
- El portafolio empieza a distinguir entre ingresos inmediatos, activos sostenibles e iniciativas de impacto.
- Cada proyecto deja documentacion, aprendizajes o relaciones que aumentan la capacidad futura del equipo.

## Preguntas para convertir en metricas

- Que queremos medir semanalmente?
- Que queremos revisar mensualmente?
- Que senales mostrarian deterioro relacional?
- Que senales justificarian invertir mas tiempo o capital en una iniciativa?
