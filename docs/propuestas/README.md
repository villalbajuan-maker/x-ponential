# Propuestas

Esta carpeta contiene ideas estructuradas para que el equipo pueda evaluarlas, discutirlas y convertirlas en accion.

Una propuesta no necesita estar perfecta para entrar aqui. Si debe tener suficiente claridad para que otra persona entienda que se quiere lograr, por que importa y cual seria el siguiente paso.

## Convencion de nombres

Usar nombres cortos y descriptivos:

`YYYY-MM-DD-nombre-de-la-propuesta.md`

Ejemplo:

`2026-06-21-plataforma-validacion-ideas.md`

## Estados sugeridos

- Borrador
- En discusion
- Aprobada
- En validacion
- Pausada
- Cerrada

Usar [TEMPLATE.md](TEMPLATE.md) como punto de partida.
