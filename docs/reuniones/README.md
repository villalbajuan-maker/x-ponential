# Reuniones

Esta carpeta guarda minutas breves de conversaciones relevantes del equipo.

## Convencion de nombres

`YYYY-MM-DD-tema.md`

Ejemplo:

`2026-06-21-reunion-fundacional.md`

## Recomendacion

La minuta debe ser corta, concreta y orientada a accion. No necesitamos transcribir todo; necesitamos preservar claridad.

Usar [TEMPLATE.md](TEMPLATE.md) como base.
